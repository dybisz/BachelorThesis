###############################################
#                Install                      #
###############################################
1) To install change directory to ./pattern_recognition.

    cd pattern_recognition

2.1) Type 'make' to compile and run Unit Tests. 
This will create a executable in pattern_recognition/bin/ 

    make

2.2) To properly install the application type 
'make install' to install the program in the default directory:

	make install

The default directory is stated in app_config.sh.

2.2.1) Installing to input directory.
Use the argument 'INSTALL_ROOT_PATH' to state full path
to installation directory:

	make install INSTALL_ROOT_PATH=~/my_programs/
	
Directory will be created if does not exist.
The INSTALL_ROOT_PATH can also be specified in app_config.sh.
Altough the argument will always override the value in the app_config.sh
